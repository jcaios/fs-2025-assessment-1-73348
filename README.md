## DublinBikes Blazor Client

### How to run
1. Run the API project (Ctrl+F5)
2. Run the Blazor client (F5)

### API Configuration
API base URL is configured in Program.cs

### Features
  Master list view with all station information
  Detail view (panel or selection)
  Search and filters (status and minimum available bikes)
  Sorting
  Paging / Load More
  Form for creating and editing stations (POST/PUT)
  Loading state and empty state
  API integration (HttpClient)
